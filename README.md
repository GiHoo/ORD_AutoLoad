원피스 랜덤 디펜스 로드 자동 불러오기 프로그램


<h3>1. exe 프로그램을 다운로드 한 후 Warcraft III\CustomMapData\ORD10에 프로그램을 넣습니다.</h3>

<h3>2. 바탕화면에 바로가기를 만든 후 exe 프로그램을 추가합니다.</h3>

<h3>3. exe 프로그램을 실행하면 자동으로 클립보드에 로드가 복사됩니다.</h3>
